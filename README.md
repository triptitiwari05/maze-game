# Maze Game

An interactive Maze Game developed using HTML, CSS, and JavaScript. Players can select different difficulty levels and navigate through the maze to reach the destination.

This Maze Game project was explored and executed as part of my web development learning journey. I studied the codebase, fixed implementation issues, and successfully ran the project to improve my understanding of HTML, CSS, and JavaScript.

## Features

- Multiple difficulty levels:
  - Easy
  - Medium
  - Hard
  - Extreme
- Interactive gameplay
- Dynamic maze generation
- Responsive user interface
- Simple and engaging design
- Start button to generate a new maze

## How to Play

1. Open the game in your browser.
2. Select a difficulty level.
3. Click the **Start** button.
4. Use the controls to navigate through the maze.
5. Reach the destination to win the game.
